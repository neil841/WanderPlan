## Getting Started

### Creating an Account

1. **Visit WanderPlan**
   - Go to [your WanderPlan URL]
   - Click **Sign Up** in the top right

2. **Fill Registration Form**
   - Enter your email address
   - Create a strong password (min. 8 characters)
   - Enter your first and last name
   - Click **Create Account**

3. **Verify Your Email**
   - Check your inbox for verification email
   - Click the verification link
   - You'll be redirected to your dashboard

**First Login Tip**: Set up your profile by clicking your avatar ‚Üí Settings

---

### Dashboard Overview

After logging in, you'll see your dashboard with:

- **My Trips**: List of all your trips (upcoming, active, past)
- **Quick Actions**: Create trip, invite collaborators
- **Recent Activity**: Latest updates across your trips
- **Filters**: Search and filter trips by tags, dates, status

## Creating Your First Trip

### Step 1: New Trip

1. Click **Create Trip** button
2. Fill in basic information:
   - **Trip Name**: "Paris Vacation"
   - **Destination**: "Paris, France"
   - **Dates**: Select start and end dates
   - **Description** (optional): Brief overview

3. **Optional Settings**:
   - Upload cover photo
   - Add tags (e.g., "europe", "family")
   - Set privacy (Private, Shared, Public)

4. Click **Create Trip**

### Step 2: Set Your Budget (Optional)

1. Click **Budget** tab
2. Enter total budget: $3,000
3. Break down by category:
   - Accommodation: $1,000
   - Transportation: $800
   - Food: $600
   - Activities: $600

4. Click **Save Budget**

## Collaboration

### Inviting Collaborators

1. Open your trip
2. Click **Collaborators** tab
3. Click **Invite Collaborator**
4. Enter email address
5. Choose permission level:
   - **Viewer**: Can view only
   - **Editor**: Can edit itinerary and expenses
   - **Admin**: Full control (except delete trip)

6. Click **Send Invitation**

The invited person receives an email with a link to accept.

### Proposals

Create professional travel proposals:

#### Creating a Proposal

1. Click **Proposals** (sidebar)
2. Click **Create Proposal**
3. Select client: Sarah Johnson
4. Add proposal details:
   ```
   Title: Paris Honeymoon Package
   Description: 7 romantic days in Paris
   Valid Until: Dec 31, 2024
   ```

5. **Add line items**:
   ```
   Round-trip flights (2 pax) | Qty: 2 | Price: $800 | Total: $1,600
   Hotel Eiffel Seine (6 nights) | Qty: 6 | Price: $250 | Total: $1,500
   Louvre Museum Tour (2 pax) | Qty: 2 | Price: $75 | Total: $150
   ```

6. **Calculate totals**:
   - Subtotal: $3,530
   - Tax (10%): $353
   - Discount: $0
   - **Total: $3,883**

7. Click **Save Draft**

- Tag trips by type: "business", "family", "solo"
- Tag by region: "europe", "asia", "domestic"
- Filter dashboard by tags

**Template Trips**:
- Create a trip as template
- Duplicate for future trips
- Adjust dates and details

---

### Collaboration

**Set Expectations**:
- Assign roles clearly (who books what)
- Use polls for group decisions
- Keep everyone updated via messages

**Designate a Lead**:
- One person as trip organizer (Admin)

**Use Ideas**:
- Collect suggestions from everyone

**Split Expenses Fairly**:

## Troubleshooting

### Can't Login

**Forgot Password**:
1. Click **Forgot Password** on login
2. Enter email address
3. Check email for reset link
4. Click link and set new password

**Email Not Verified**:
1. Check spam folder
2. Click **Resend Verification** on login
3. Verify email before logging in

---

### Events Not Saving

**Check Connection**:
- Verify internet connection
- Look for red error message
- Try refreshing page

**Browser Issues**:
- Clear cache and cookies
- Try different browser
- Disable browser extensions

---

### Collaborator Issues

**Invitation Not Received**:
1. Check spam folder
2. Verify email address is correct
3. Resend invitation
4. Try different email

**Can't Edit Trip**:
- Check permission level (need Editor or Admin)
- Contact trip owner to update role
- Refresh page

---

### Budget Not Calculating

**Mismatch**:
- Verify all expenses have category
- Check for duplicate expenses
- Ensure dates are correct

**Split Not Working**:
- Verify all collaborators added
- Check split amounts add up to total
- Recalculate

---

### PDF Export Failing

**Large Trips**:
- Limit date range
- Use Compact format
- Exclude photos

**Browser Timeout**:
- Try different browser
- Clear cache
- Contact support

---

### Calendar Sync Issues

**Not Syncing**:
1. Disconnect and reconnect Google account
2. Check calendar permissions
3. Verify selected calendar exists
4. Try manual sync

**Duplicate Events**:
- Disconnect sync
- Delete duplicate events in Google Calendar
- Reconnect with "Replace" option
