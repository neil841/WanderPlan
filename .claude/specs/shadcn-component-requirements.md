## Custom Components to Build

These are NOT available from shadcn and must be custom-built:

1. **TimePicker** - Time selection input
   - Recommendation: Build with `input` + custom dropdown or use a library like `react-time-picker`

2. **DateRangePicker** - Extend `calendar` component to support range selection
   - Recommendation: Modify shadcn `calendar` component

3. **DraggableEventCard** - Drag-and-drop wrapper
   - Recommendation: Use `@dnd-kit` with shadcn `card`

4. **LocationAutocomplete** - Extend `command` with Nominatim API
   - Recommendation: Wrap shadcn `command` with API integration

5. **CurrencyInput** - Currency-aware number input
   - Recommendation: Extend shadcn `input` with currency formatting
